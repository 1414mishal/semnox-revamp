# Semnox Website Revamp

A full redesign concept of [semnox.com](https://www.semnox.com), rebuilt as a fast static site with a dark editorial design system.

- 17 pages covering every section of the original site: solutions (Parafait, Tixera, Delikopos), company, media, careers, and contact
- Tailwind CSS (CDN), Lucide icons, Inter, scroll reveal and counter animations
- Mobile-first with a full-screen menu and touch friendly dropdowns
- Rebuild pages with `node src/build.mjs`

Live: https://1414mishal.github.io/semnox-revamp/
